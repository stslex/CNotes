# CNotes
The simple Android Application - Notes on compose

- Jetpack Compose
- Jetpack compose navigation
- Koin with koin navigation
- Material 3, dynamic colors, adaptive icon
- Multimodules
- MVVM
- Coroutines, Flow
- Room
- Jetpack paging library
